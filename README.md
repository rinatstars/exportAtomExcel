
# exportExcel

Утилита автоматического экспорта результатов анализа из **Atom** в **Excel**.

Программа запускается автоматически из системы **AutoExport** Atom после завершения анализа, обрабатывает XML‑отчёты и формирует таблицы Excel с результатами измерений.

---

# Назначение

`exportExcel` предназначен для:

- автоматической обработки XML‑отчётов Atom
- извлечения результатов анализа
- формирования Excel‑таблиц
- подготовки данных для последующей обработки

Программа используется совместно с:

- **Atom**
- **ExportExcelConfigurator**

---

# Как работает система

1. Atom завершает анализ
2. Atom сохраняет XML‑отчёт в папку:

```
Atom\Reports
```

3. Atom запускает:

```
exportExcel.exe
```

4. Программа:

- находит новый XML
- извлекает данные анализа
- формирует Excel файл

# Основные функции

## Автоматический экспорт

Программа автоматически:

- обрабатывает XML отчёты
- извлекает элементы анализа
- формирует Excel таблицу

---

## Настраиваемая структура Excel

Можно задать:

- порядок элементов
- формат таблицы
- формат чисел
- дополнительные поля

Настройка производится через:

```
ExportExcelConfigurator.exe
```

---

# Конфигурация

Основные параметры находятся в:

```
settings.json
```

Пример:

```json
{
    "xml_folder": "C:\\Atom 3.3 x64\\Reports",
    "excel_folder": "C:\\Atom 3.3 x64\\Reports",
    "elements": [
        "Si",
        "Fe",
        "Al"
    ]
}
```

## Параметры

### xml_folder

Папка, в которой Atom сохраняет XML отчёты.

```
Atom\Reports
```

### excel_folder

Папка для сохранения Excel файлов.

### elements

Список элементов, которые будут включены в таблицу.

---

# Настройка Atom

В файле:

```
Atom\Ini\DB.ini
```

должны быть параметры:

```
[AutoExport]
dir=Atom\Reports\
programPath=Atom\Reports\scripts\exportExcel.exe
bXML=1

[dbSettings]
autoExport=1
```

---

# Запуск

Обычно программа **не запускается вручную**.

Она запускается автоматически системой **AutoExport** Atom.

Ручной запуск возможен для тестирования:

```
exportExcel.exe
```
